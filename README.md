# districtapi-node

Official Node.js SDK for [districtapi.dev](https://districtapi.dev) — US school district and school data API.

## Installation

```bash
npm install districtapi
# or
yarn add districtapi
```

## Status

This SDK is currently a placeholder. The API and full SDK are in development.

## What It Does

Once released, this SDK will provide a simple Node.js client for looking up US school districts and schools by address, coordinates, or NCES ID.
